Vercel 部署指南

把 AI 广告素材生成平台发布到公网，生成可发给老师的 demo 链接

目标：获得类似 https://ai-ad-creative-platform.vercel.app/image/create 的链接，老师可直接打开体验产品。

准备工作

  • GitHub 账号（代码已在仓库中）
  • 约 10 分钟时间
  • 无需信用卡（Vercel + Neon 免费套餐够用）

第一步：注册 Vercel

  1. 打开 https://vercel.com/signup
  2. 选择 Continue with GitHub，用 GitHub 账号登录
  3. 授权 Vercel 访问你的 GitHub 仓库

第二步：导入项目

  1. 登录后点击 Add New… → Project
  2. 在列表中找到 ai-ad-creative-platform，点击 Import
  3. Framework 会自动识别为 Next.js，保持默认即可

第三步：创建数据库（Neon PostgreSQL）

Vercel 上不能使用本地 SQLite，需要免费云数据库：

  1. 在 Vercel 项目配置页，找到 Storage 或 Integrations
  2. 搜索并添加 Neon（Postgres）
  3. 创建免费数据库，Vercel 会自动填入 DATABASE_URL

或在 neon.tech 单独注册，复制连接字符串，手动添加到环境变量。

第四步：配置环境变量

在 Vercel 项目 Settings → Environment Variables 中添加：

变量名值说明
DATABASE_URL Neon 自动提供 PostgreSQL 连接
JWT_SECRET 任意长随机字符串 例如 my-demo-secret-2026-xyz
STORAGE_DIR /tmp/storage Vercel 临时文件目录

OPENAI_API_KEY 和 REPLICATE_API_TOKEN 可不填，会自动使用 Mock 演示模式。

第五步：部署

  1. 点击 Deploy
  2. 等待 2～5 分钟，看到 Congratulations! 即成功
  3. 复制 Vercel 给你的域名，例如：
    https://ai-ad-creative-platform-xxx.vercel.app

第六步：发给老师的链接

产品首页：

https://你的域名.vercel.app

图片创作页（推荐）：

https://你的域名.vercel.app/image/create

演示账号：

邮箱：demo@ad-creative.local
密码：demo123456

常见问题

问题解决
部署失败，报数据库错误 检查 DATABASE_URL 是否已配置，格式为 postgresql://...
打开页面但登录失败 确认 JWT_SECRET 已设置，重新 Deploy 一次
生成图片后看不到图 演示环境使用 Mock 模式，刷新后可能消失（Vercel 临时存储限制），UI 交互正常即可
localhost 和老师打不开的链接 localhost 只有你能访问；老师必须用 vercel.app 链接
